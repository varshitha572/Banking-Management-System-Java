# Banking Management System

## Project Overview
A console-based Banking Management System developed using Java, JDBC, and MySQL.

## Features
- User Login Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- Transaction History
- Input Validation
- Database Connectivity using JDBC

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub

## Database Tables

### users
- id
- username
- password
- balance

### transactions
- id
- user_id
- type
- amount
- date

## Concepts Implemented
- JDBC Connectivity
- PreparedStatement
- SQL Queries
- Exception Handling
- CRUD Operations
- Database Testing
